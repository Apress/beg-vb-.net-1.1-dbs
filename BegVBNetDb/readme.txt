Hi and welcome to the source code for Beginning VB .NET Databases book from Apress.

To get this code up and running, please follow these instructions.

- Follow the database installation instructions given in Chapter 1 of the book to install MSDE and the sample databases this code uses.
- Make sure the code is unzipped to C:\BegVBNetDb
- When working with a chapter conerning ASP.NET (C10 and 11), the home directory of your web browser should point to the directory containing the code for the chapter.
- Don't forget that Chapter 22 uses .NET 2.0 code and will not open in Visual Studio .NET 2002 or 2003.
